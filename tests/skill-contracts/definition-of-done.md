# Skill contract

Una skill MVP debe cumplir:

- `SKILL.md` valido.
- `name` igual al nombre de carpeta.
- `description` especifica.
- Secciones `Use`, `Workflow`, `Output`, `Checklist`.
- Sin dependencias Claude-only.
- `agents/openai.yaml` con prompt que mencione `$skill`.
- Fixture o ejemplo realista asociado.
